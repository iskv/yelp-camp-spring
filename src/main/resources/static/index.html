About YelpCamp Web App

Current functionality:
  1. User registration
  2. User authorization
  3. Adding / editing / deleting user's campground
  4. Adding / editing / deleting user's comment per campground

Technical Details

The project used:
  1. Java
  2. Spring (Spring Boot, Spring Security, Spring Data JPA), Thymeleaf
  3. PostgreSQL
  4. Bootstrap, Font Awesome, Google fonts
This web-application is designed with RESTful.
